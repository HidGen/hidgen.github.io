# hidgen.github.io

## 🏋 Считалка для качалки

Лёгкий офлайн-дневник тренировок (PWA). Открывается по ссылке, ставится из браузера —
**без App Store и APK**.

**Ссылка:** https://hidgen.github.io/liftlog/

### Как поставить — iPhone (Safari)
1. Открой ссылку **в Safari**.
2. «Поделиться» (квадрат со стрелкой внизу) → **«На экран „Домой"»** → «Добавить».
3. Запускай иконку **«Считалка»** с домашнего экрана.

> Важно: на iPhone именно Safari. У приложений с домашнего экрана данные ещё и
> не вытесняются системой (в обычной вкладке Safari могут стереться через ~неделю).

### Как поставить — Android (Chrome)
1. Открой ссылку в Chrome.
2. Баннер **«Установить приложение»** (или меню ⋮ → «Установить приложение»).
3. Иконка появится на экране.

### Что попробовать и прислать в фидбек
- Заведи тренировку → упражнение (попробуй **силовое** и **кардио**) → запиши пару подходов.
- Удобно ли логировать «в пару тапов» усталыми руками? Что неочевидно/бесит?
- Глянь Историю; попробуй **⬇ Экспорт** (файл-бэкап).

### Про данные
История хранится **только на устройстве** (офлайн). Бэкап: История → **⬇ Экспорт**.
Удалишь иконку — данные уйдут (синхронизация между устройствами будет позже).
