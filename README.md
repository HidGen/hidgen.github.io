# 🏋 Считалка для качалки — гайд для тестировщика

Лёгкий дневник тренировок. Это веб-приложение (PWA): открывается по ссылке, ставится
из браузера — **без App Store и магазинов приложений**. Данные сохраняются и
синхронизируются автоматически, ничего настраивать не нужно.

**Ссылка:** https://hidgen.github.io/liftlog/

---

## Установка (по желанию — можно просто пользоваться в браузере)
- **iPhone (Safari):** открой ссылку → «Поделиться» (квадрат со стрелкой) → **«На экран „Домой"»**.
- **Android (Chrome):** открой ссылку → баннер **«Установить приложение»** (или меню ⋮ → «Установить приложение»).

Так удобнее: открывается как отдельное приложение, на iPhone ещё и данные не вытесняются системой.
Если приложение уже стоит, а обновления не видно — закрой и переоткрой иконку (подтянется свежая версия).

## Как устроено: две роли
Роль переключается в **⚙ Настройки**:
- 🏋 **Качок** — ведёшь свои тренировки.
- 📋 **Тренер** — добавляешь подопечных, смотришь их историю и можешь вести их тренировку со своего устройства.

У каждого пользователя есть короткий **ID** (виден в ⚙ Настройки). Качок даёт свой ID
тренеру — после этого тренер видит его тренировки. Синхронизация — автоматическая,
работает и офлайн (долетит, когда появится сеть).

## Если ты — Качок 🏋
1. **«Начать тренировку»** (кнопка внизу).
2. **«+ Упражнение»** → выбери тип (Силовое / Свой вес / Кардио / На время) и название
   (тапни из недавних или впиши своё), задай стартовый вес/параметры.
3. На экране упражнения жми большую **«Записать N × R»** после каждого подхода.
   Следующий подход по умолчанию = прошлый, поэтому часто хватает **одного тапа**;
   число можно тапнуть и ввести точно (вес, повторы, время, пульс…).
4. **«Готово»** → упражнение попадёт в список. Затем **«Завершить тренировку»**.
5. Прошлые тренировки — прямо на главном экране (раскрываются) и в **«📋 История»**.
6. Свой **ID** для тренера — в ⚙ Настройки (кнопка «копировать»).

## Если ты — Тренер 📋
1. ⚙ Настройки → выбери роль **«📋 Тренер»** → закрой настройки крестиком (✕).
2. На стартовом экране — список подопечных. Добавь подопечного кнопкой **＋** вверху
   (нужен его ID, который он тебе дал) — можно указать имя.
3. **Тапни подопечного** → откроется его история тренировок и кнопка
   **«▶ Тренировка за …»** — можно вести его тренировку со своего телефона
   (удобно, когда ему самому несподручно записывать).
4. **⟳** вверху — обновить данные всех подопечных.
5. Назад к списку — **«← подопечные»**; сменить роль обратно на качка — ⚙ Настройки.

## Что хочется услышать в фидбеке
- Удобно ли записывать тренировку «в пару движений» усталыми руками? Что неочевидно или раздражает?
- Поток тренера: понятно ли добавлять/выбирать подопечного и вести тренировку за него?
- Любые баги, тормоза, странности отображения — особенно на твоём конкретном телефоне.
- Чего не хватает для реального использования в зале?

## Про данные и приватность
Приложение **не спрашивает и не хранит персональных данных** — только тренировки.
История привязана к твоему ID на устройстве и в облаке. Это тестовая версия,
синхронизация пока демо-уровня (для сбора фидбека).
