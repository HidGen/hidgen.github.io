🏋 Считалка для качалки

Лёгкий дневник тренировок. Открывается по ссылке, ставится из браузера — без App Store и магазинов приложений. Всё сохраняется и синхронизируется само, настраивать ничего не нужно.

▶ Открыть приложение

📲 Поставить как приложение

По желанию — можно просто пользоваться в браузере.

iPhone (Safari): открой ссылку → «Поделиться» (квадрат со стрелкой ↑) → «На экран „Домой"».
Android (Chrome): открой ссылку → баннер «Установить приложение» (или меню ⋮ → «Установить приложение»).

Если приложение уже стоит, а нового не видно — закрой и снова открой иконку.

Две роли

Роль переключается в ⚙ Настройки:

🏋 Качок ведёшь свои тренировки. 📋 Тренер ведёшь тренировки подопечных и смотришь их прогресс.

У каждого есть короткий ID (в ⚙ Настройки). Качок даёт свой ID тренеру — и тренер видит его тренировки. Работает и без интернета (синк догонит позже).

Если ты — Качок 🏋

  1. Нажми «Начать тренировку» (внизу экрана).
  2. «+ Упражнение» → выбери тип (Силовое / Свой вес / Кардио / На время) и название (тапни из недавних или впиши своё), задай стартовый вес/параметры.
  3. После каждого подхода жми большую кнопку «Записать N × R». Следующий подход по умолчанию такой же — часто хватает одного тапа; число можно тапнуть и ввести точно.
  4. «Готово» → упражнение в списке. Затем «Завершить тренировку».
  5. Прошлые тренировки — на главном экране и в «📋 История».
  6. Свой ID для тренера — в ⚙ Настройки (кнопка «копировать»).

Если ты — Тренер 📋

  1. ⚙ Настройки → роль «📋 Тренер» → закрой настройки крестиком (✕).
  2. На стартовом экране — список подопечных. Добавь по кнопке ＋ вверху (нужен ID, который тебе дал качок) — можно указать имя.
  3. Тапни подопечного → откроется его история и кнопка «▶ Тренировка за …»: можно вести его тренировку со своего телефона.
  4. ⟳ вверху — обновить данные подопечных.

Что хочется в фидбеке

  • Удобно ли записывать «в пару движений» усталыми руками? Что неочевидно/раздражает?
  • Тренеру: понятно ли добавлять/выбирать подопечного и вести за него?
  • Любые баги, тормоза, странности — особенно на твоём телефоне.
  • Чего не хватает для реального зала?

Приложение не спрашивает и не хранит персональных данных — только тренировки. Это тестовая версия для сбора отзывов. Спасибо, что тестируешь! 🙏
